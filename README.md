# 🍡 Kirby Mochi Recepy

A cute, Kirby-themed single-page website showcasing a delicious homemade mochi recipe.

Built with **Vite + React + TypeScript**, this project focuses on simplicity, responsiveness, and playful design using a pink Kirby-inspired theme.

---

## ✨ Features

- 🎀 Kirby-inspired UI (pink theme + playful font)
- 📱 Fully responsive (desktop → mobile)
- 🧁 Clean recipe layout:
  - Ingredients section
  - Step-by-step procedure
  - Image display

- ⚡ Fast build with Vite
- 🚀 Deployed via GitHub Pages

---

## 🧱 Tech Stack

- ⚛️ React
- 🔷 TypeScript
- ⚡ Vite
- 🎨 CSS (no frameworks)
- 🌐 GitHub Pages (deployment)

---

## 🚀 Getting Started

### 1. Clone the repo

```
git clone https://github.com/YOUR_USERNAME/kirby-mochi-recepy.git
cd kirby-mochi-recepy
```

---

### 2. Install dependencies

```
npm install
```

---

### 3. Run locally

```
npm run dev
```

Then open:

```
http://localhost:5173/
```

---

## 🛠️ Build

```
npm run build
```
